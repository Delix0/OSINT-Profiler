## 🔧 API

### DataManager

```python
from core.data_manager import DataManager

dm = DataManager()

# Создать
target_id = dm.create_target({...})

# Получить
target = dm.get_target(target_id)

# Обновить
dm.update_target(target_id, {...})

# Удалить
dm.delete_target(target_id)

# Поиск
results = dm.search_targets("query")
```

### ReportGenerator

```python
from generator import ReportGenerator

gen = ReportGenerator()

# Один отчёт
path = gen.generate_report(target_id)

# Все отчёты
paths = gen.generate_all_reports()
```

Больше примеров в **EXAMPLES.md**!

## 🔒 Безопасность

⚠️ **Важные напоминания:**

1. Все данные хранятся локально в `data/database.json`
2. **НЕ** коммитьте реальные данные в Git!
3. Добавьте в `.gitignore`:
   ```
   data/database.json
   output/*.html
   ```
4. HTML-отчёты содержат чувствительную информацию
5. Храните отчёты в безопасном месте
